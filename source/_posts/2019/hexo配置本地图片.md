# 安装hexo-asset-image
```$xslt
在hexo目录执行
npm install https://github.com/CodeFalling/hexo-asset-image --save
```

# 使用
```
hexo n "git配置sshkey"
```
此时可以看到在目录下创建了一个 git配置sshkey 文件和 git配置sshkey 文件夹

```$xslt
在md文档中输入即可
![](git配置sshkey/gitlib-sshkey.png)
```
